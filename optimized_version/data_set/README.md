# 甘薯数据集

## （1）**数据集简介**
data_set 共 1255 幅。  
TrainVal 用于训练和验证（训练：验证 = 6:2），占比为 8/10，共 1004 幅；  
Test 用于测试，占比2/10，共 251 幅。

```
data_set
    ├── TrainVal
    │     ├── FY    发芽，共 356 幅。
    │     ├── MF    霉腐，共 160 幅。
    │     ├── SS    损伤，共 260 幅。
    │     └── ZC    正常，共 228 幅。
    │
    └── Test 
          ├── FY    发芽，共 89 幅。
          ├── MF    发芽，共 40 幅。
          ├── SS    发芽，共 65 幅。
          └── ZC    发芽，共 57 幅。
```
## （2）**使用步骤**

- 模型训练时会将 TrainVal 文件夹中的图像划分为训练集和验证集，验证集在 TrainVal 中的占比为 2:8 ，即 0.25 。

- 模型训练之后进行保存，即可加载 Test 中的图像进行测试。
